Ajoute un utilisateur

D’après la documentation d’Apple

  • Pomme > Réglages Système > Utilisateurs et groupes

  • “Ajouter un compte”, (sous la liste des utilisateurs à droite)

  • Saisissez le mot de passe

  • Cliquez sur le menu local “Nouveau compte”, puis choisissez “Administrateur”

  • Nom du nouvel utilisateur : faire simple

  • Mot de passe et indice

  • Cliquez sur Créer l’utilisateur.

Ouvre le terminal

Tout ce qu’il faut savoir sur le terminal

  • Cmd + Espace pour ouvrir “Spotlight search”
  • Taper “Terminal” dans la barre de recherche

Installe Homebrew

  • Visitez la page de Homebrew
  • Installez le ainsi : /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Installe R et Rstudio

  • R : brew install r
  • Rstudio : brew cask install rstudio
  • Pour les mettre à jour si besoin : brew update & brew upgrade

Configure Rstudio

Vérifie si Git est installé

  • which git
  • S’il ne l’est pas : brew install git

Présente toi à Git

  • git config --global user.name 'Jane Doe'
  • git config --global user.email 'jane@example.com
  • git config --global --list
