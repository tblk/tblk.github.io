Images

Utiliser ImageMagick pour convertir, compresser et transformer des images
Manipuler

Sur cette page

  • Convertir une image
  • Compresser une image
  • Passer en nuances de gris

Convertir une image

convert image.jpg image.png

Compresser une image

convert image.jpg[50%x50%] -quality 70 compressed_image.jpg

Passer en nuances de gris

Ici, on utilise le package magick, qui rend accessible depuis R la librairie ImageMagick.

gray_frog = magick::image_read(
  "https://upload.wikimedia.org/wikipedia/commons/5/5a/Hyla_japonica_sep01.jpg"
  ) |> 
  magick::image_quantize(colorspace = "gray")
gray_frog
